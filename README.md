Shrimp Disease Detection using YOLOv8

A real-time computer vision system using Ultralytics YOLOv8 to detect and classify shrimp diseases (e.g., WSSV, White Gut) to improve aquaculture productivity. Built with Python, OpenCV, and deep learning for early outbreak detection


# Project Overview
This project implements an automated, real-time solution for detecting shrimp diseases in aquaculture environments using the Ultralytics YOLOv8 object detection framework. Early diagnosis is critical in preventing the rapid spread of viral infections like White Spot Syndrome Virus (WSSV), which can cause high mortality rates in ponds.

Key Features: 

  Real-Time Detection: Rapidly identifies diseased shrimp in images and video streams.
  
  High Accuracy: Optimised for common aquatic challenges like floating foam and water movement.
  
  Lightweight Deployment: Suitable for resource-constrained platforms in actual production settings.
  
  Multi-Class Classification: Specifically trained to recognise abnormal behaviours such as curling, leaping, or death.
